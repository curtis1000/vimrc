# vimrc
My Vim Configuration

Prerequisites
- Install [Powerline Fonts](https://github.com/powerline/fonts)
- Install [Vundle](https://github.com/VundleVim/Vundle.vim)

Installation
- clone this repository to `~/.vim`
- `cp ~/.vim/.vimrc ~/.vimrc`
- mkdir ~/.vim-swap
- Run `:PluginInstall` and/or `:PluginUpdate`
